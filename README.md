# Malazan Companion

Personal, spoiler-safe reading database for Kyle's read-through of *Malazan: Gardens of the Moon*.

## Goal

Capture characters, locations, factions, events, relationships, and open questions from Kyle's own reading progress. Input can start as Hermes-assisted page photos; output should be queryable via SQLite/CLI and readable in Obsidian.

## Core rules

- Use only Kyle-provided pages/notes unless Kyle explicitly asks for outside research.
- Avoid spoilers beyond the saved reading position.
- Store structured notes and short provenance snippets, not long OCR dumps from copyrighted pages.
- Mark uncertainty instead of pretending the book is clearer than it is.
