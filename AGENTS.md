# Malazan Companion Agent Instructions

This project is Kyle's personal spoiler-safe reading database for Malazan.

## Non-negotiables

- Do not use external Malazan wiki/summaries/fandom knowledge unless Kyle explicitly asks for outside-spoiler research.
- Use only Kyle-provided page photos, OCR text, and notes for database updates.
- Do not store long copyrighted passages from the book. Store structured facts, short summaries, page/chapter references, and very short quote snippets only when needed.
- Preserve uncertainty: use confidence scores and open questions.
- Every stored fact should be tied to a source/page/photo when possible.
- Prefer local-first files: SQLite database plus Obsidian Markdown exports.
